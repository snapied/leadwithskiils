6-Week Intensive Curriculum

Each week builds on the previous, culminating in a complete technical portfolio

Week 1

How Software is Built (The Big Picture)

Goal: Understand how every piece of an app fits together

What You'll Learn:

  • The Internet fundamentals (HTTP, HTTPS, DNS, CDNs)
  • Client-Server architecture and request-response cycles
  • Frontend vs Backend vs Database roles
  • Mobile app architecture (iOS/Android)
  • Web browsers and how they render pages
  • Version control systems (Git basics)
  • Development environments (staging, production)

Hands-On Projects:

  • Tech Stack Mapping: Draw the complete architecture of Netflix (from mobile app to recommendation engine)
  • User Journey Mapping: Trace what happens when you order food on Swiggy (every technical step)
  • Performance Analysis: Compare loading speeds of 3 popular apps and identify bottlenecks

✅ Outcome:

You'll confidently explain how any app works to stakeholders and identify technical constraints early

Week 2

Mastering Data & SQL (The Language of Decisions)

Goal: Stop waiting for engineers to pull data. Get it yourself.

What You'll Learn:

  • Database fundamentals (Tables, Rows, Columns, Keys)
  • SQL vs NoSQL databases (when to use each)
  • Writing SELECT, WHERE, GROUP BY, JOIN queries
  • Data aggregation and filtering techniques
  • Database performance and indexing basics
  • Data warehousing concepts (ETL processes)
  • Common database tools (MySQL, PostgreSQL, MongoDB)

Hands-On Projects:

  • Customer Analysis: Write SQL to find top 10 customers by revenue (using real e-commerce dataset)
  • Cohort Analysis: Calculate monthly user retention rates using SQL
  • A/B Test Analysis: Query experiment results to determine statistical significance
  • Dashboard Creation: Build a simple analytics dashboard with key metrics

✅ Outcome:

You'll independently pull data, create reports, and make data-driven product decisions without waiting for analysts

Week 3

APIs & System Integration (How Systems Talk)

Goal: Design products that connect seamlessly with other tools and services

What You'll Learn:

  • REST APIs vs GraphQL vs WebSockets
  • HTTP methods (GET, POST, PUT, DELETE)
  • API authentication (API keys, OAuth, JWT)
  • Webhooks and real-time data sync
  • API rate limiting and error handling
  • Third-party integrations (Stripe, Twilio, SendGrid)
  • Microservices architecture basics

Hands-On Projects:

  • Live API Integration: Connect Slack to Google Sheets using Postman (real data flow)
  • Payment Gateway Setup: Test Stripe API for processing payments
  • Notification System: Build SMS alerts using Twilio API
  • Integration Roadmap: Design API strategy for a multi-platform product

✅ Outcome:

You'll confidently scope integration projects, estimate timelines, and troubleshoot API issues with engineering teams

Week 4

Cloud Infrastructure & Scaling (Where Products Live)

Goal: Make informed decisions about hosting, scaling, and infrastructure costs

What You'll Learn:

  • Cloud platforms comparison (AWS, Azure, GCP)
  • Server types (shared, VPS, dedicated, serverless)
  • Content Delivery Networks (CDNs) and caching
  • Load balancing and auto-scaling
  • Database scaling (vertical vs horizontal)
  • Security fundamentals (SSL, firewalls, encryption)
  • DevOps basics (CI/CD, deployment pipelines)

Hands-On Projects:

  • Cost Analysis: Calculate hosting costs for 1M vs 10M users
  • Build vs Buy Decision: Compare building in-house vs using SaaS for key features
  • Scaling Strategy: Design infrastructure roadmap for rapid growth
  • Security Audit: Identify vulnerabilities in a sample application

✅ Outcome:

You'll make smart technical investments, predict scaling challenges, and optimize infrastructure costs

Week 5

AI & Machine Learning Integration (Beyond the Hype)

Goal: Build realistic AI features based on actual capabilities and limitations

What You'll Learn:

  • How Large Language Models (LLMs) actually work
  • Machine Learning types (supervised, unsupervised, reinforcement)
  • AI APIs (OpenAI, Google AI, AWS AI services)
  • Data requirements for ML models
  • AI ethics and bias considerations
  • Prompt engineering and fine-tuning
  • AI product metrics and evaluation

Hands-On Projects:

  • AI Feature Pitch: Design and prototype an AI-powered feature for an existing product
  • Chatbot Implementation: Build a customer service bot using OpenAI API
  • Recommendation Engine: Design a content recommendation system
  • AI ROI Analysis: Calculate the business impact of implementing AI features

✅ Outcome:

You'll separate AI hype from reality and build practical AI features that deliver measurable business value

Week 6

Technical Leadership & Interview Mastery (Putting It All Together)

Goal: Lead technical discussions and ace your next technical PM interview

What You'll Learn:

  • Software Development Lifecycle (SDLC) and Agile methodology
  • Technical estimation techniques (story points, planning poker)
  • Code review process and quality assurance
  • Technical debt management and prioritization
  • Cross-functional team communication
  • Technical interview frameworks and question types
  • Portfolio presentation and storytelling

Hands-On Projects:

  • Technical PRD: Write a detailed product requirements document with technical specifications
  • Sprint Planning: Lead a mock sprint planning session with engineering estimates
  • Portfolio Presentation: Create a compelling presentation of all your projects
  • Mock Interview: Live technical PM interview with personalized feedback

✅ Outcome:

You'll confidently lead technical discussions, estimate project timelines, and showcase your skills in any technical PM interview
